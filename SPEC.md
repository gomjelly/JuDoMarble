# JuDoMarble - 프로젝트 스펙

## 개요
블루마블 / 모두의마블 스타일의 보드게임을 실제 테이블 위에 프로젝션(투사)하는 시뮬레이션 소프트웨어.

## 기술 스택
- 언어: C++
- IDE: Visual Studio 2022
- UI 프레임워크: Qt 6.8.3

## 핵심 기능

### 1. 디스플레이 선택
- 출력할 모니터 / 프로젝터 디스플레이를 선택하는 기능
- 멀티 디스플레이 환경 지원

### 2. 편집 화면 (메인 에디터)
- 가운데 캔버스 영역에 다각형(polygon)을 배치 및 편집
- 프로젝션 매핑 툴과 유사한 UX
  - 다각형 추가 버튼들 (사각형, 삼각형, 자유형 등)
  - 꼭짓점 드래그로 다각형 변형
  - 이동, 크기 조절, 회전 지원

### 3. 시뮬레이션 모드
- "시뮬레이션" 버튼을 누르면 선택한 디스플레이에 편집한 다각형들이 출력됨
- 편집 화면과 출력 화면 분리

## 향후 추가될 기능 (단계적 구현)
- 보드게임 칸 데이터 (땅, 이벤트, 세금 등)
- 플레이어 말 이동 애니메이션
- 주사위 굴리기
- 게임 규칙 및 UI
